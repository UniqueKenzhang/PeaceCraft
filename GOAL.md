# ⚖️ PeaceCraft (天平之祭) - 核心愿景 (The North Star)

龙王大人圣谕：**“我们的初始目标是要设计一个类似 Anno (纪元) 的建设游戏。”**

## 🏗️ Anno 式设计核心准则 (Anno-style Pillars)
1. **产业链优先 (Production Chains)**：资源不是凭空产生的，必须由基础资源经过建筑加工产出高级物资（如：小麦 -> 面粉 -> 补给品）。
2. **人口阶层 (Population Tiers)**：不同等级的人口有不同的需求，只有满足了他们（如：粮食、娱乐、搞怪武器），文明才能升级。
3. **平衡与管理 (Balance & Management)**：在满足生产的同时，必须通过补给战场来维持双界的和平天平。
4. **视觉布局感 (Visual Sense of Place)**：即使是 Web 版，也要通过 UI 营造出一种在经营庞大帝国的感觉。

## ⚠️ 开发者自律手册
- **每次动工前**：必须阅读本文件，对齐当前开发任务是否符合《Anno》式的建设感。
- **每次提交后**：在汇报中说明本功能如何增强了“建设”与“经营”体验。
