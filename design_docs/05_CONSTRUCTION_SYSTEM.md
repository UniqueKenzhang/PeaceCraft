# 模块三：Anno风格建设与经济系统 (Anno-style Construction & Economy)

这是玩家后方基地的核心玩法。与传统Anno游戏不同，玩家需要同时管理和发展**“人类帝国”**与**“深渊领地”**两个截然不同的国度。两者共同为前线提供“三色物资”，并维系玩家总体的经济稳定。

## 1. 核心任务与目标

- **双边发展**: 为人类与深渊建立两条独立、特色鲜明且互相依赖的产业链。
- **非对称建设**: 提供两套完全不同的建筑、人口和需求系统。
- **跨界物流**: 鼓励玩家为了生产高级物资，在两个国度之间建立复杂的物流网络。
- **统一经济**: 在拥有独立资源和人口的同时，维持一个统一的财政系统，并设定全局性的“经济破产”失败条件。

---

## 2. 新增核心机制：双国度管理 (Dual-Nation Management)

- **独立地图**: 玩家拥有两个独立的建设地图，可通过UI随时切换视角。
- **共享国库，独立仓储**:
    - **共享国库**: 两个国度的所有财政收支（税收、维护费）都汇入一个统一的国库。经济破产是全局性的。
    - **独立仓储**: 两个国度拥有各自独立的仓库和资源库存。人类无法直接使用深渊仓库里的物资，反之亦然。
- **跨界运输**:
    - 游戏进行到中期，玩家可解锁特殊的“跨界传送门”或“走私船坞”建筑。
    - 玩家可在此类建筑中设定运输路线，将一个国度的物资运送到另一个国度，但会产生一定的花费或时间延迟。这是生产高级“恶搞物资”的必要前提。

---

## 3. 功能拆解与实现方案 (已更新)

### 任务 3.1：非对称的资源与产业链

- **阵营独占资源**:
    - **人类帝国**: 拥有农田、森林、温顺的牲畜。产出如“小麦”、“木材”、“棉花”。
    - **深渊领地**: 拥有硫磺矿、真菌丛林、可奴役的魔物。产出如“硫磺”、“暗影菇”、“魂能水晶”。

- **非对称产业链**:
    - 两个国度拥有完全不同的生产建筑和配方，即使是同类产品，其生产链也完全不同。
    - **示例 - 基础食物**:
        - **人类**: `农田` -> `小麦` -> `磨坊` -> `面粉` -> `面包房` -> `面包`
        - **深渊**: `真菌洞穴` -> `暗影菇` -> `腐化池` -> `菌类肉糜`
    - **高级物资的依赖关系**:
        - **示例 - 高级恶搞物资“糖衣炮弹”**:
            - 需要人类方的`糖`和`铁壳`。
            - 需要深渊方的`魂能炸药`作为核心。
            - 玩家必须先将`糖`和`铁壳`从人类帝国运到深渊领地，才能在深渊的“诡计工坊”中合成该物资。

### 任务 3.2：非对称的人口、建筑与物流

- **独立的人口阶层**:
    - **人类**: `农民` -> `市民` -> `学者`。需求偏向秩序、信仰和舒适（食物、教堂、酒馆）。
    - **深渊**: `苦工` -> `监工` -> `祭司`。需求偏向服从、恐惧和力量（菌肉、拷问室、献祭坑）。
    - 两套人口系统独立升级，拥有各自不同的幸福度指标。

- **独立的建筑风格与功能**:
    - 所有建筑，包括住宅、生产设施、公共服务设施，都拥有完全不同的视觉风格和名称。

### 任务 3.3：统一的财政系统与失败条件

- **财政模型**:
    - **统一核算**: 财政报表会分别列出人类和深渊的收入与支出，但最终汇总计算总盈余和总国库。
    - **收入**: 来自两个国度各自的人口税收。
    - **支出**: 两个国度所有建筑的维护费总和。

- **经济失败条件**:
    - **全局破产**: 当**总国库**为负数，并且在之后的一段时间内**总盈余**仍为负，则判定为“经济破产”，游戏结束。
    - **单边崩溃的风险**: 即使财政健康，若一个国度因人口流失而完全停摆，将导致高级物资链断裂，玩家也难以继续游戏。

## 4. 关联模块接口 (已更新)

- **输入**:
    - `(来自模块四)` 接收“逃兵”作为特殊劳动力。**联盟逃兵自动加入人类帝国，深渊逃兵自动加入深渊领地。**
- **输出**:
    - `(至模块二)` 提供玩家**两个国度仓库中**“三色物资”的总库存信息，供玩家交付。
    - `(至UI系统)` 提供两套独立的人口、幸福度数据，以及一个统一的财政数据用于界面展示。
