# PeaceCraft 游戏系统与 UI 交互白皮书 (v4.6)

## 1. 核心视觉体系 (Visual Paradigm)
*   **设计风格**: 帝国模拟 (Anno 1800 Style) 结合 奇幻博弈。
*   **史诗背景 (Epic Backdrop)**: 全屏动态缩放的麦田与剑原画，象征“和平”与“工艺”的永恒主题。
*   **沉浸式滤镜**: 带有电影级暗角 (Vignette) 与动态毛玻璃 (Backdrop Blur) 效果。

## 2. 关键 UI 要素与功能说明

### A. 帝国总署 (Imperial Command Sidebar) - [左侧面板]
*   **功能**: 核心政务与建设中心。
*   **领地缩略图**: 顶部动态显示当前场景（人类帝国/恶魔领地）的英雄画作，实时反馈统治疆域。
*   **人口阶层 (Population Tiers)**: 
    *   **人类**: 农民 (Peasants) -> 市民 (Citizens)。
    *   **恶魔**: 奴隶 (Slaves) -> 领主 (Lords)。
*   **幸福度计量表 (Happiness Meters)**: 实时监控各阶层的满意度，影响资源产出。
*   **建筑工部 (Construction Grid)**: 
    *   **工业生产**: 部署磨坊、锯木厂、采石场等生产建筑。
    *   **市政民生**: 部署提升幸福度与人口的城市建筑。

### B. 战略物资调拨舰队 (Logistic Fleet) - [底部中央]
*   **功能**: 实现后方基地与前线战场的资源输送，取代了传统的进度条操作。
*   **交互逻辑**:
    1.  **点击装载**: 直接点击三艘不同功能的运输舰（铁甲壹号、游吟商船、皇家重载）完成物资装载。
    2.  **视觉反馈**: 舰船进入 `READY` 状态，产生金色流光反馈。
    3.  **授权启程**: 点击浮现的“授权商队启程”勋章，触发派遣逻辑。
    4.  **往返机制**: 舰船经历 `DISPATCHING` (派遣) 与 `RETURNING` (返航) 阶段，带来策略性的冷却时间感。

### C. 万界天平 · 中央战场 (Battle View) - [中央图层]
*   **功能**: 展示世界势力的博弈状态。
*   **能量平衡条 (Balance Bar)**: 实时反映联盟与深渊的势力对比。
*   **VS 对抗张力**: 巨大的图标（🛡️ vs 🔥）通过动态缩放展现前线的激烈程度。
*   **战地通信流 (War Log)**: 实时滚动播放前线物资抵达及战况变化的通信文本。

### D. 帝国状态栏 (Top Status Bar) - [顶部]
*   **场景切换器**: 在“联盟旧世界”与“深渊新世界”间无缝切换。
*   **资源监控器**: 实时显示当前场景的核心资源储备（粮/木/石 或 肉/骨/灵）。
*   **和平指数 (Peace Index)**: 游戏的核心终极目标数值，受前线博弈结果影响。

## 3. 游玩逻辑闭环 (Gameplay Loop)
1.  **建设与积累**: 在人类或恶魔场景中规划建筑，提升人口与资源产量。
2.  **幸福度维护**: 确保生产链闭环，提升阶层幸福度以获取更高级物资。
3.  **战略调拨**: 通过底部舰队，将积累的物资（如真钢兵刃或文化产物）有节奏地派遣至前线。
4.  **平衡博弈**: 在中央战场观察物资抵达后的天平变化，确保势力平衡或达成特定胜利条件。
5.  **循环演进**: 随着和平指数的提升，解锁更复杂的生产链与更强大的远征舰船。
