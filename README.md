# ⚖️ PeaceCraft (天平之祭) - 开发蓝图

**龙王大人御制：双生国度管理与和平博弈史诗。**
**核心愿景：一个类似 Anno (纪元) 的深度建设游戏。**

## 🛠️ 五大工程包开发进度

### 工程包一：双界基座 (Core World Logic)
- [x] **1.1 多场景同步框架**：人类、魔族、战场 100% 效率同步。
- [x] **1.2 场景瞬移交互**：Tab 切换无缝跳转。
- [x] **1.3 资源位扩展**：木材、石材、白骨、魔能等基础资源。

### 工程包二：生存与建设 (Anno Style - 深度打磨)
- [x] **2.1 人类生产链**：农场 -> 伐木场 -> 采石场。
- [x] **2.2 魔族生产链**：腐肉池 -> 祭坛 -> 裂隙。
- [x] **2.3 人口驱动逻辑**：人口基数决定资源产出乘数。
- [x] **2.4 人口阶层系统**：农民/市民、奴隶/领主。
- [x] **2.7 4x4 格子建造**：初步实现网格化精准建设。
- [x] **2.16 沉浸式地图交互重构**：Anno 式地图预览、右键拆除、动态落成动画。
- [ ] **2.13 地块肥力系统**：实装肥沃地块逻辑与视觉特效。
- [ ] **2.15 复合产业链**：实现面包、精钢等高阶资源转化。

### 工程包三：战备与真假货 (博弈强化)
- [x] **3.1 物资填装 UI**：比例滑块系统。
- [x] **3.2 武器数值模型**：真货稳健，假货崩盘逻辑。
- [x] **3.3 运输延迟机制**：3秒在途模拟。
- [ ] **3.4 阶层限制武备**：高级武器需对应人口阶层才能产出。

### 工程包四：战场与搞怪机制 (视觉与反馈)
- [x] **4.1 平衡仪表盘**：动态力量天平 UI。
- [x] **4.2 逃兵转化逻辑**：士兵触发搞怪反馈并转化为人口。
- [x] **4.3 演兵画面**：方阵位移与危险预警。
- [ ] **4.4 资源产出气泡**：实装建筑上方的动态产出反馈。

### 工程包五：危机与闭环 (Game Cycle)
- [x] **5.1 末日倒计时**：失衡触发自毁结局。
- [x] **5.2 帝国派发令**：远方帝国随机摊派事件。
- [x] **5.3 和平循环联调**：首个完整 Demo 跑通。
- [ ] **5.4 资源看板面板**：一屏掌控双界供需平衡。
- [ ] **5.5 本地存档功能**：LocalStorage 进度保存。
