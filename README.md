# ⚖️ PeaceCraft (天平之祭)

龙王大人御制：一款关于平衡、搞怪与和平的单机 Web 建设博弈游戏。

## 📜 核心价值观
本游戏旨在通过玩家的建设与物资调配，维持人类与魔族的力量平衡，最大限度减少伤亡。利用“搞怪武器”让士兵逃离战场并转化为领地人口，是实现长远繁荣的唯一途径。

---

## 🛠️ 任务清单 (Task List)

### ✅ 第一阶段：基础架构 (已完成)
- [x] **1.1 场景基座**：人类、魔族、战场三地无缝切换。
- [x] **1.2 单机引擎**：全逻辑 JS 化，实现人口、资源、战力每秒同步更新。
- [x] **1.3 补给系统**：实现“真/假/搞怪”三滑块联动，支持实时补给结算。
- [x] **1.4 逃兵转化**：搞怪货比例直接决定战场逃兵回流至领地的人口数。

### 🚧 第二阶段：搞怪趣味与闭环 (进行中)
- [ ] **2.1 搞怪对白系统**：为不同搞怪武器设计专属的战场日志文本（如：“魔族踩到了惊吓地雷，大喊‘妈妈我要回家’并溜走了”）。
- [ ] **2.2 住房与限制**：人口不能无限增长，需消耗资源建造“民舍/洞穴”来承载逃兵。
- [ ] **2.3 视觉差异化**：强化人类（蓝色/秩序）与魔族（红色/混沌）的视觉对比。

### 📅 第三阶段：平衡压力与终局 (待启动)
- [ ] **3.1 毁灭倒计时**：天平偏离 70% 时触发末日警告音效与全屏滤镜。
- [ ] **3.2 本地存档**：利用浏览器缓存实现“存档/读档”功能。
- [ ] **3.3 成就系统**：统计大人的“和平拯救人数”，颁发勋章。
